Welcome to Electro DM!

Electro DM is a tool to make DM'ing more engaging for players and easier on the DM.
  1. Configure default human players in Player Config.
  2. Build and progress through encounters in Encounter Mode.
    • Players get a cool visual display of initiative (and other optional modes)!
    • Navigate to {{clientUrl}} on a TV, iPad, or another browser to see the player view. Show this to your players when playing!
    • Notice that as you progress through initiative order in Encounter Mode, the display is updated to show the current player's turn.
  3. Switch between 'modes' on the client display using Scene Manager.
    • Bonus! You can use an Elgato Stream Deck to automate switching between scenes.
  4. Take notes that are saved in your browser using Notes.
Disclaimer: Electro DM is very much still a work in process. Things might change rapidly. Additionally, I'm working on finding open-source resources. Currently, some proprietary DnDBeyond resources are embedded which will need to be removed.
